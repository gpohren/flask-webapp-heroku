## Deploying a Web App with Flask and Heroku

Disponibilizar um simples Web App com Python e Flask no Heroku

## Ferramentas utilizadas

* Python
* Flask
* Heroku

## Criar os secrets

* HEROKU_API_KEY
* HEROKU_APP_NAME
* HEROKU_EMAIL
